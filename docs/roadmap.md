# Roadmap

## MVP (actual)

- Login JWT con roles.
- CRUD de grupos y estudiantes.
- Módulos y seed inicial.
- Calendario y sesiones de clase.
- Asistencia por sesión.
- Pagos con métodos y pagos parciales.
- Caja del día con desglose por método.
- Gastos (admin).
- Flujo de caja básico (admin).
- Tiempo real con Socket.IO.

## Siguientes pasos

- Reportes avanzados y exportación.
- Gestión completa de usuarios y permisos desde UI.
- Comprobantes de pago con subida de archivos.
- Auditoría visible en UI.
- Notificaciones de deuda.
